# Практика 2 — Задание 5
## Симулятор кофейного аппарата

## Описание
Консольная программа симулирует работу кофемашины с напитками «Американо» и «Латте», учетом ресурсов и выручки.

## Возможности
- Ввод начальных запасов воды и молока.
- Приготовление напитков по рецептам.
- Проверка доступности ресурсов.
- Итоговая статистика по продажам и доходу.

## Запуск
Из папки `sem1/practice/2 раб тетр/5`:

```bash
dotnet restore Program.csproj
dotnet run --project Program.csproj
```
