# Практика 2 — Задание 6
## Симуляция «Бактерии vs Антибиотик»

## Описание
Консольная модель роста бактерий по часам с учетом снижающейся эффективности антибиотика.

## Возможности
- Ввод начального количества бактерий и дозировки.
- Пошаговый вывод динамики по часам.
- Итоговый анализ эффективности.
- Прогноз роста бактерий при прекращении действия антибиотика.

## Запуск
Из папки `sem1/practice/2 раб тетр/6`:

```bash
dotnet restore Program.csproj
dotnet run --project Program.csproj
```
