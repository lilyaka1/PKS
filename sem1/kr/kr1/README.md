# Контрольная работа 1 (1 семестр)
## Календарь с событиями

## Описание
Консольное приложение-календарь с поддержкой пользовательских событий и повторений.

## Возможности
- Вывод календаря на год.
- Вывод календаря на выбранную дату.
- Добавление событий с частотой повторения:
  - без повторения;
  - ежедневно;
  - еженедельно;
  - ежемесячно;
  - ежегодно.
- Просмотр событий на конкретный день.
- Проверка дня недели (будний/выходной).

## Файл проекта
- `calendar.cs` — весь код приложения.

## Запуск
Из папки `sem1/kr/kr1`:

```bash
dotnet script calendar.cs
```

Если `dotnet-script` не установлен, можно создать консольный проект и перенести файл `calendar.cs` в `Program.cs`.
