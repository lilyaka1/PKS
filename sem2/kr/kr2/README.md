# Контрольная работа 2 (2 семестр)

## Анализатор сетевых подключений (Avalonia)

## Описание

Кроссплатформенное desktop-приложение на Avalonia для анализа сетевых интерфейсов компьютера и проверки URL/URI.

## Функциональные возможности

- Получение списка сетевых интерфейсов через `System.Net.NetworkInformation`.
- Отображение параметров интерфейса:
  - IP-адреса;
  - маски подсети (IPv4);
  - MAC-адрес;
  - статус подключения;
  - скорость;
  - тип интерфейса.
- Анализ URL/URI:
  - схема;
  - хост;
  - порт;
  - путь;
  - query-параметры;
  - fragment.
- Дополнительно:
  - ping хоста;
  - DNS-резолвинг;
  - определение типа адреса (локальный, публичный, loopback);
  - история проверок URL с сохранением на диск.

## Стек технологий

- C# / .NET 9
- Avalonia UI 11

## Структура проекта

- `MainWindow.axaml` — интерфейс приложения.
- `MainWindow.axaml.cs` — логика анализа интерфейсов, URL, ping, DNS, история.
- `App.axaml`, `App.axaml.cs` — конфигурация приложения.
- `Program.cs` — точка входа.
- `NetworkAnalyzer.csproj` — файл проекта.

## Запуск

Из папки `sem2/kr/kr2`:

```bash
dotnet restore NetworkAnalyzer.csproj
dotnet run --project NetworkAnalyzer.csproj
```

## Сборка

```bash
dotnet build NetworkAnalyzer.csproj
```
