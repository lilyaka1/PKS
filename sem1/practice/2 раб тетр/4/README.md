# Практика 2 — Задание 4
## Игра «Угадай число»

## Описание
Консольная игра, где программа угадывает число пользователя в диапазоне 0..63, используя двоичное представление и серию вопросов.

## Возможности
- Пошаговое угадывание на основе битов.
- Валидация ответов пользователя.
- Объяснение алгоритма после завершения.

## Запуск
Из папки `sem1/practice/2 раб тетр/4`:

```bash
dotnet restore Program.csproj
dotnet run --project Program.csproj
```
