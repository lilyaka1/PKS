# Контрольная работа 2 (1 семестр)
## Калькулятор матриц

## Описание
Консольное приложение для ввода матриц и вычисления обратной матрицы.

## Возможности
- Ввод размеров матриц.
- Заполнение матриц:
  - вручную;
  - случайными числами в диапазоне.
- Вывод матриц в табличном виде.
- Вычисление обратной матрицы для квадратных матриц.
- Расчет определителя (рекурсивно через миноры).

## Ограничения
- Обратная матрица вычисляется только для квадратных матриц с ненулевым определителем.
- Рекурсивный расчет определителя неэффективен для очень больших матриц.

## Файлы
- `Program.cs` — основная логика.
- `MatrixCalculator.csproj` — файл проекта.

## Запуск
Из папки `sem1/kr/kr2`:

```bash
dotnet restore MatrixCalculator.csproj
dotnet run --project MatrixCalculator.csproj
```

## Сборка
```bash
dotnet build MatrixCalculator.csproj
```
