# Практика 2 — Задание 2
## Проверка счастливого билета

## Описание
Консольная программа определяет, является ли шестизначный билет счастливым: сумма первых трех цифр должна быть равна сумме последних трех.

## Возможности
- Проверка корректности ввода.
- Подробный разбор цифр и сумм.
- Демонстрационные примеры счастливых и обычных билетов.

## Запуск
Из папки `sem1/practice/2 раб тетр/2`:

```bash
dotnet restore Program.csproj
dotnet run --project Program.csproj
```
