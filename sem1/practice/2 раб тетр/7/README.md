# Практика 2 — Задание 7
## Планировщик колонизации Марса

## Описание
Программа рассчитывает максимальную толщину защитного слоя модулей при размещении на ограниченном поле.

## Возможности
- Проверка размещения модулей в двух ориентациях.
- Бинарный поиск максимальной толщины защиты.
- Анализ вариантов размещения и использования площади.

## Запуск
Из папки `sem1/practice/2 раб тетр/7`:

```bash
dotnet restore Program.csproj
dotnet run --project Program.csproj
```
