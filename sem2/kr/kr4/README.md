# Контрольная работа 4 (2 семестр)

## Веб-приложение «Туристический путеводитель» (ASP.NET Core MVC)

## Описание

Веб-приложение на ASP.NET Core MVC, представляющее собой интерактивный туристический путеводитель по городам России.

## Функциональные возможности

- Просмотр списка городов с кратким описанием.
- Поиск городов по названию.
- Просмотр подробной информации о городе:
  - Название, регион, население.
  - История города.
  - Список достопримечательностей.
- Просмотр информации о достопримечательности:
  - Название, описание, история.
  - Часы работы.
  - Стоимость посещения.

## Модель данных

- **City** — город:
  - Id, Name, Region, Population.
  - Description, History.
  - ImageUrl, CoatOfArmsUrl.
  - Связь с достопримечательностями.

- **Attraction** — достопримечательность:
  - Id, Name, Description, History.
  - ImageUrl, WorkingHours, TicketPrice.
  - Связь с городом (CityId).

## Стек технологий

- C# / .NET 9
- ASP.NET Core MVC
- Entity Framework Core
- SQLite

## Структура проекта

- `Models/` — сущности City и Attraction.
- `Data/` — контекст базы данных TouristDbContext.
- `Controllers/` — контроллеры CitiesController и AttractionsController.
- `Views/` — Razor-представления.
- `wwwroot/images/` — изображения.
- `Program.cs` — точка входа и конфигурация.

## Запуск

Из папки `sem2/kr/kr4/TouristGuideWeb`:

```bash
dotnet restore TouristGuideWeb.csproj
dotnet run
```

Приложение будет доступно по адресу: http://localhost:5000

## Сборка

```bash
dotnet build TouristGuideWeb.csproj
```

## База данных

При первом запуске автоматически создается база данных SQLite с демонстрационными данными:
- 4 города (Москва, Санкт-Петербург, Казань, Владивосток).
- 10 достопримечательностей.
