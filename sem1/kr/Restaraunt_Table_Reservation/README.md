# Контрольная работа 4 (1 семестр)
## Система бронирования столиков в ресторане

## Описание
Консольное приложение для авторизации пользователей и управления бронированием столиков.

## Возможности
- Авторизация и регистрация пользователей.
- Разделение по ролям пользователей.
- Главное меню после входа в систему.
- Подготовленная структура для меню бронирований и столиков.
- Хранение данных через EF Core.

## Технологии
- C# / .NET
- Entity Framework Core
- SQLite (локальная БД)

## Структура
- `Program.cs` — точка входа, инициализация БД.
- `Data/` — контекст БД.
- `Models/` — модели (пользователи, столы, бронирования).
- `Services/` — сервисы авторизации/логики.
- `UI/` — консольные меню.
- `RestaurantConsoleApp.csproj` — файл проекта.

## Запуск
Из папки `sem1/kr/Restaraunt_Table_Reservation`:

```bash
dotnet restore RestaurantConsoleApp.csproj
dotnet run --project RestaurantConsoleApp.csproj
```

## Примечание
Часть UI-модулей находится в стадии доработки, но структура проекта готова для расширения.
