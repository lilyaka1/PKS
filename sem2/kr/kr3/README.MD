# Контрольная работа 3 (2 семестр)

## HTTP-монитор (Avalonia)

## Описание

Кроссплатформенное desktop-приложение на Avalonia для мониторинга HTTP-запросов и истории браузера.

## Функциональные возможности

- Получение истории браузера (Chrome, Firefox).
- Мониторинг HTTP-запросов в реальном времени.
- Детальная информация о запросах:
  - HTTP-метод.
  - URL.
  - Код ответа.
  - Время выполнения.
  - Размер данных.
- Фильтрация по браузеру.
- Сохранение истории в файл.
- Возможность очистки истории.

## Стек технологий

- C# / .NET 9
- Avalonia UI 11

## Структура проекта

- `MainWindow.axaml` — интерфейс приложения.
- `MainWindow.axaml.cs` — логика мониторинга HTTP.
- `App.axaml`, `App.axaml.cs` — конфигурация приложения.
- `Program.cs` — точка входа.
- `HttpMonitorWpf.csproj` — файл проекта.

## Запуск

Из папки `sem2/kr/kr3`:

```bash
dotnet restore HttpMonitorWpf.csproj
dotnet run --project HttpMonitorWpf.csproj
```

## Сборка

```bash
dotnet build HttpMonitorWpf.csproj
```
