# Практика 2 — Задание 1
## Ряд Маклорена для e^x

## Описание
Консольная программа вычисляет значение функции `e^x` через ряд Маклорена с заданной точностью и сравнивает результат с `Math.Exp(x)`.

## Возможности
- Ввод `x` и `epsilon`.
- Суммирование ряда до достижения точности.
- Расчет абсолютной погрешности.
- Отдельный расчет n-го члена ряда.

## Запуск
Из папки `sem1/practice/2 раб тетр/1`:

```bash
dotnet restore Program.csproj
dotnet run --project Program.csproj
```
