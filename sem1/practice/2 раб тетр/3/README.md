# Практика 2 — Задание 3
## Сокращение дроби

## Описание
Программа приводит дробь `M/N` к несократимому виду с использованием алгоритма Евклида для НОД.

## Возможности
- Проверка валидности числителя/знаменателя.
- Обработка отрицательных значений.
- Вывод сокращенной дроби и десятичного значения.
- Демонстрационные примеры.

## Запуск
Из папки `sem1/practice/2 раб тетр/3`:

```bash
dotnet restore Program.csproj
dotnet run --project Program.csproj
```
