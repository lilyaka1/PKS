# Приложение для управления библиотекой (Library Manager)

## Описание

Это полнофункциональное приложение для управления библиотекой, разработанное на **C#** с использованием **WPF** (Windows Presentation Foundation) и **Entity Framework Core**. Приложение позволяет управлять книгами, авторами и жанрами в локальной базе данных SQLite.

## Требования

- .NET 6.0 или выше
- Visual Studio 2022 (или VS Code с расширением C#)
- SQLite (встроен в EF Core)

## Основные возможности

### 1. Управление книгами

- Просмотр всех книг в DataGrid
- Добавление новой книги
- Редактирование информации о книге
- Удаление книги (с подтверждением)
- Поиск книг по названию
- Фильтрация по автору
- Фильтрация по жанру
-  Отображение количества книг в наличии

### 2. Управление авторами

- Просмотр всех авторов
- Добавление нового автора (имя, фамилия, дата рождения, страна)
- Редактирование информации об авторе
- Удаление автора (с проверкой наличия книг)
- Защита от удаления автора, у которого есть книги

### 3. Управление жанрами

- Просмотр всех жанров
- Добавление нового жанра (название, описание)
- Редактирование жанра
- Удаление жанра (с проверкой наличия книг)
- Защита от удаления жанра с книгами

## Модель данных

### Сущность Book (Книга)

```
- Id (int) - первичный ключ
- Title (string) - название (макс. 255 символов)
- ISBN (string) - ISBN код (макс. 20 символов)
- PublishYear (int) - год публикации
- QuantityInStock (int) - количество в наличии
- AuthorId (int) - внешний ключ на автора
- GenreId (int) - внешний ключ на жанр
```

### Сущность Author (Автор)

```
- Id (int) - первичный ключ
- FirstName (string) - имя (макс. 100 символов)
- LastName (string) - фамилия (макс. 100 символов)
- BirthDate (DateTime) - дата рождения
- Country (string) - страна происхождения (макс. 100 символов)
```

### Сущность Genre (Жанр)

```
- Id (int) - первичный ключ
- Name (string) - название жанра (макс. 100 символов)
- Description (string) - описание (макс. 500 символов)
```

## Связи между сущностями

- **Book → Author**: Многие-к-одному (ManyToOne)

  - Одна книга связана с одним автором
  - Один автор может иметь много книг
  - Каскадное удаление: при удалении автора удаляются его книги
- **Book → Genre**: Многие-к-одному (ManyToOne)

  - Одна книга принадлежит одному жанру
  - Один жанр может содержать много книг
  - Каскадное удаление: при удалении жанра удаляются его книги

## примеры использования

### Добавление новой книги:

1. Нажмите кнопку "Добавить"
2. Заполните поля: Название, ISBN, Год, Количество
3. Выберите автора и жанр из выпадающих списков
4. Нажмите "Сохранить"

### Поиск книги:

1. Введите название (или часть названия) в поле "Поиск по названию"
2. Нажмите кнопку "Поиск"
3. Для очистки фильтра нажмите "Очистить"

### Фильтрация по автору:

1. Выберите автора из выпадающего списка
2. Нажмите кнопку "Фильтровать"

### Удаление книги:

1. Выберите книгу из таблицы
2. Нажмите кнопку "Удалить"
3. Подтвердите удаление

## Обработка ошибок

- Валидация данных перед сохранением в БД
